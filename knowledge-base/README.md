## 🎯 项目定位和价值

### 解决的核心问题

**AI 测试 vs 传统测试的本质差异**:

```
传统测试: 输入 → 确定性输出 → 对比预期结果
AI 测试:  输入 → 概率性输出 → 评测判定 → 判定结果
```

### 核心创新价值

1. **三文件分离架构**：将非结构化对话转化为可量化、可追溯的工程化资产
2. **可追溯性优先**：证据链完整，问题定位精确
3. **配置中心化**：业务逻辑与配置数据彻底分离
4. **动态Prompt组装**：支持运行时配置和扩展

## 🏗️ 核心架构亮点

### 三文件分离架构

```
测试用例模板      执行记录         评测结果        测试报告
   (意图)         (证据)          (判决)         (汇总)
   (before)       (during)        (after)        (summary)
```

**核心洞察**："执行记录是证据，评测结果是判决。可追溯比自动化更重要。"

### 10维度评测体系

| 维度 | 权重 | 评测重点 |
|------|------|----------|
| **信息准确性** | 25% | 提供的信息是否准确无误 |
| **合规性** | 20% | 是否符合业务规则和法律法规 |
| **安全性** | 15% | 是否泄露敏感信息或存在安全风险 |
| **专业性** | 10% | 回答是否专业、得体 |
| **响应速度** | 5% | 响应时间是否在可接受范围内 |
| **用户体验** | 10% | 回答是否易于理解和使用 |
| **问题解决能力** | 15% | 是否有效解决用户问题 |

## 🔧 技术实现深度

### 动态Prompt组装引擎

```python
class EvaluatorPromptAssembler:
    """评测Prompt动态组装引擎 - 按需加载维度规则，节省Token"""

    def assemble(self, test_case: dict, dimension: str) -> str:
        """组装评测Prompt：基础模板 + 维度规则 + 场景注入"""
        base_prompt = self.template_loader.load("customer-service-evaluator.md")
        dimension_section = self.template_loader.load_section(f"{dimension}-focus.md")
        return self._merge(base_prompt, dimension_section, test_case)
```

### 配置注册中心

```python
class ConfigRegistry:
    """配置注册中心 - 全局唯一、不可变、显式初始化"""

    @classmethod
    def initialize(cls, config_dir: str = None, scenario: str = None):
        """显式初始化配置注册中心"""
        loader = ConfigLoader(config_dir=config_dir)
        instance = cls(loader, scenario=scenario)
        cls._instance = instance
        return instance
```

### 模块化设计

每个组件职责单一，支持独立测试和替换：

```python
# 可插拔的评测维度路由
def route_evaluation(test_case: dict, dimension: str) -> str:
    """根据维度动态选择评测规则片段"""
    section_map = {
        "multi": "multi-dimension-focus.md",
        "multi_turn": "multi-turn-focus.md",
        "prompt_injection": "prompt-injection-rules.md",
    }
    section_file = section_map.get(dimension)
    if section_file:
        return template_loader.load_section(section_file)
    return None
```
