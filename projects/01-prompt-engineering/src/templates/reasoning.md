# 推理能力测试 Prompt 模板

## 目的

测试大模型的逻辑推理能力，使用思维链（Chain-of-Thought）方法。

## 模板

```
你是一个逻辑推理专家。请逐步分析以下问题。

问题：{problem}

请按以下步骤进行分析：

步骤1：理解问题
- 仔细阅读问题
- 识别关键信息
- 明确问题目标

步骤2：分析已知条件
- 列出所有已知条件
- 理解条件之间的关系

步骤3：应用逻辑推理
- 选择合适的推理方法
- 逐步推导结论
- 验证每一步的正确性

步骤4：得出结论
- 总结推理过程
- 给出最终答案
- 解释推理依据

最终答案：[你的答案]
```

## 使用示例

**输入**：
```
problem: "如果所有的猫都是动物，所有的动物都需要食物，那么所有的猫都需要食物吗？"
```

**预期输出**：
```
步骤1：理解问题
- 问题要求判断"所有猫都需要食物"这个结论是否正确
- 涉及两个前提条件的逻辑推理

步骤2：分析已知条件
- 前提1：所有的猫都是动物
- 前提2：所有的动物都需要食物
- 需要判断：所有猫是否需要食物

步骤3：应用逻辑推理
- 猫属于动物的子集（前提1）
- 动物集合中的所有成员都需要食物（前提2）
- 因此，猫作为动物的子集，也需要食物

步骤4：得出结论
- 推理过程符合逻辑
- 结论成立

最终答案：是的，所有的猫都需要食物。
```

## 评估指标

- **准确性**：答案是否正确
- **逻辑性**：推理过程是否合理
- **完整性**：是否包含所有步骤
- **清晰度**：表达是否清晰易懂
