# 实验 001：Prompt 变体对比

## 🎯 实验目标

研究不同 Prompt 结构对大模型响应质量的影响。

## 🧪 实验方法

### 测试场景
1. 逻辑推理问题（10个测试用例）
2. 数学应用题（10个测试用例）
3. 常识推理（10个测试用例）

### Prompt 变体

#### 变体 1：直接提问
```
解决这个问题：{problem}
```

#### 变体 2：分步引导
```
请分步解决这个问题：
{problem}
```

#### 变体 3：Few-shot 示例
```
这里有一些示例：
示例1：...
示例2：...

现在请解决：{problem}
```

#### 变体 4：思维链（CoT）
```
[使用 reasoning.md 模板]
```

### 评估指标
- **BLEU 分数**：与参考答案的相似度
- **人工评估**：1-5 分质量评分
- **准确性**：答案是否正确

## 📅 实验时间线

- Day 1：设计 Prompt 和测试用例
- Day 2：运行实验
- Day 3：分析结果
- Day 4：撰写实验报告

## 📊 预期结果
