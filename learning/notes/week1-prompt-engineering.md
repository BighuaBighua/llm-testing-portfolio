### Day 1: Prompt 基础（2026-04-01）

#### 什么是 Prompt？
- Prompt 是用户给大模型的输入文本或指令
- 核心作用：引导模型生成符合期望的输出

#### Prompt 的基本结构
1. **角色设定**：定义 AI 的角色
2. **任务描述**：明确要做什么
3. **输出格式**：指定输出形式
4. **示例**：提供参考案例

#### 学习心得
- Prompt 质量直接影响模型输出质量
- 清晰、具体的 Prompt 效果更好
- 需要不断实验和优化
