# 大模型测试工程师课程总览

## 📚 课程结构

本课程共分为 **5 个阶段**，预计学习周期 **10-12 周**。

---

## 第一阶段：Prompt 工程（第 1-2 周，面试核心考点）

### Week 1: Prompt 入门

**总目标**：掌握 Prompt 基础设计逻辑，能独立撰写符合评测要求的基础 Prompt

**每日安排**：
- 周一：Prompt 基础（角色/Token/上下文）→ 输出学习笔记
- 周二：角色设定技巧 → 输出 5 类 AI 角色模板
- 周三：Few-shot 示例学习 → 输出 3 个实战案例
- 周四：思维链（CoT）→ 输出逻辑推理对比报告
- 周五：Prompt 迭代优化 → 输出优化案例文档
- 周末：复盘整理，完成 AI 客服评测 Prompt 初稿

**本周固定产出**：
- ✅ Prompt 学习笔记
- ✅ 5 类 AI 角色模板
- ⏳ 3 个 Few-shot 实战案例
- ⏳ 逻辑推理对比报告
- ⏳ 优化案例文档
- ⏳ AI 客服评测 Prompt 初稿

---

### Week 2: Prompt 进阶 + 安全测试

**总目标**：掌握高级 Prompt 技巧与对抗测试方法，能独立搭建 AI 评测测试用例体系

**每日安排**：
- 周一：复杂任务拆解 → 输出多轮对话评测案例
- 周二：批量测试数据生成 → 输出 100 条客服测试题 CSV
- 周三：Prompt 注入攻击 → 输出 5 个攻击测试用例
- 周四：模型安全防御测试 → 输出敏感话题测试报告
- 周五：偏见公平性测试 → 输出模型倾向性分析报告
- 周末：完成项目 1：AI 客服评测方案（简历核心）

**本周固定产出**：
- ⏳ 多轮对话评测案例
- ⏳ 100 条客服测试题 CSV
- ⏳ 5 个攻击测试用例
- ⏳ 敏感话题测试报告
- ⏳ 模型倾向性分析报告
- ⏳ 完整 AI 客服评测方案

---

## 第二阶段：Python 编程 + 数据处理（第 3-5 周，实操核心）

### Week 3: Python 基础强化

**总目标**：夯实 Python 基础，能独立编写基础数据处理与测试脚本

**每日安排**：
- 周一 - 周五：语法/循环/函数/文件读写/异常处理 → 输出 20+ 练习脚本、10 个实用函数
- 周末：完成数据格式转换小工具

**本周固定产出**：
- ⏳ 20+ 基础练习脚本
- ⏳ 10 个实用函数
- ⏳ 数据格式转换小工具

---

### Week 4: 大模型 API 调用实战

**总目标**：熟练调用大模型 API，实现基础自动化评测能力

**每日安排**：
- 周一：requests 库基础 → 公开 API 调用练习
- 周二 - 周三：大模型 API 对接 → 输出 Hello World 脚本、JSON 解析脚本
- 周四 - 周五：批量 API 调用 + 重试机制 → 输出自动化评测工具 V1
- 周末：完成 50 条问题批量评测，输出结果 CSV

**本周固定产出**：
- ⏳ API 调用练习脚本
- ⏳ Hello World 脚本
- ⏳ JSON 解析脚本
- ⏳ 自动化评测工具 V1
- ⏳ 50 条问题批量评测结果 CSV

---

### Week 5: pandas 数据分析

**总目标**：掌握评测数据处理与可视化能力，能生成专业评测报告

**每日安排**：
- 周一 - 周三：数据筛选/分组/聚合 → 输出评测数据分析脚本
- 周四 - 周五：matplotlib 可视化 + 报告生成 → 输出评测图表 + 完整报告
- 周末：重构工具，完成自动化评测工具 V2（GitHub 开源版）

**本周固定产出**：
- ⏳ 评测数据分析脚本
- ⏳ 数据可视化图表
- ⏳ 完整评测报告
- ⏳ 自动化评测工具 V2

---

## 第三阶段：大模型理论 + 评估方法论（第 6-7 周，面试问答核心）

### Week 6: 大模型基础

**总目标**：吃透大模型核心基础概念，从容应对面试理论提问

**每日安排**：
- 周一：Transformer 基础（注意力机制）→ 输出概念笔记
- 周二：Token/上下文窗口 → 输出 Token 计算工具
- 周三：Fine-tuning/RAG/Agent 对比 → 输出选型对比表
- 周四：主流大模型特性 → 输出国内模型选型指南
- 周五：模型幻觉/局限 → 输出风险测试清单

**本周固定产出**：
- ⏳ 大模型基础概念笔记
- ⏳ Token 计算工具
- ⏳ 技术选型对比表
- ⏳ 国内模型选型指南
- ⏳ 模型风险测试清单

---

### Week 7: 评估指标 + 统计学

**总目标**：掌握大模型科学评估方案，能独立设计 A/B 测试

**每日安排**：
- 周一：准确率/F1 值等基础指标 → 输出指标计算脚本
- 周二：NLP 专用指标（BLEU/ROUGE）→ 输出指标计算代码
- 周三：人工评估量表设计 → 输出评估标准文档
- 周四：A/B 测试设计 → 输出模型对比方案
- 周五：统计学基础 → 输出显著性分析脚本

**本周固定产出**：
- ⏳ 指标计算脚本/代码
- ⏳ 人工评估标准文档
- ⏳ A/B 测试设计方案
- ⏳ 显著性分析脚本
- ⏳ 垂直领域评测框架
